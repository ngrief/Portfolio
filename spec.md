# Nathaniel Trief Portfolio Website

## Overview
A professional, multi-section portfolio website for Nathaniel Trief featuring modern design, responsive layout, and interactive elements.

## Core Features

### Navigation & Layout
- Fixed navigation bar with smooth scrolling to sections: About, Skills, Projects, Data Visualization Gallery, Contact
- Dark/light mode toggle functionality
- Responsive design that works on desktop, tablet, and mobile devices
- Modern styling with gradient accents and smooth animations

### Hero Section
- Profile photo display using IMG_3024.JPG
- Name and professional title
- Social media badges/links
- Eye-catching introduction with animated elements

### About Section
- Professional bio and background information
- Personal story and career highlights
- Skills and expertise overview

### Skills Section
- Technical skills display with visual indicators
- Categorized skill sets (e.g., programming languages, frameworks, tools)
- Interactive skill cards or progress bars

### Projects Section
- Dynamic project cards with hover effects
- Each project includes:
  - Project title and description
  - Technologies used
  - Links to GitHub repository
  - Links to live demo (when available)
  - Project screenshots or thumbnails

### Data Visualization Gallery Section
- Interactive gallery showcasing data visualization projects
- Grid layout with hover animations and lightbox viewing functionality
- Featured visualizations include:
  - VotesbyParty.png - Election Analysis visualization
  - Bio.png - Microbial Diversity Dashboard
  - seasonality.png - California Wildfire Seasonality Analysis
  - Tab.png - NYC Citi Bike Tableau Dashboard
  - Uber dash.png - NYC Uber Analytics Dashboard
- Each visualization includes:
  - Project title and short description
  - Links to GitHub repository or live demo when available
  - Hover effects for enhanced interactivity
  - Lightbox modal for full-size viewing
- Responsive grid layout that adapts to different screen sizes

### Contact Section
- Contact form for inquiries
- Direct contact information (email, phone)
- Social media links
- Professional networking links

### Footer
- Contact information summary
- Last updated date
- Copyright information
- Additional social links

## Technical Requirements
- Built with React for component-based architecture
- Styled with Tailwind CSS for responsive design
- Smooth scroll behavior between sections
- Hover animations and transitions
- Mobile-first responsive design
- Dark/light theme switching with persistent preference
- Lightbox functionality for image viewing in Data Visualization Gallery

## Data Storage
- Static content and project information stored in frontend
- Data visualization images and metadata stored in frontend
- No backend data persistence required
- Theme preference stored in browser localStorage
